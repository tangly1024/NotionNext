# NotionNext 文档导航

本目录用于帮助开源贡献者和新开发者快速理解项目并安全协作。

## 文档目标

- 降低新贡献者的接入门槛
- 统一项目协作流程，减少重复沟通
- 避免个性化配置误提交导致冲突
- 让 PR 评审聚焦业务代码而非环境差异

## 与现有文档关系

- 根目录 `README.md`：项目介绍与展示
- 根目录 `CONTRIBUTING.md`：对外贡献入口
- 根目录 `DEVELOPMENT.md`：开发工具与命令
- 本目录文档：补齐“架构 + 配置 + 协作”细节
